# Abrar Islam - Personal Portfolio Website

🚀 **Backend Developer | Angular Developer | Transitioning to React Full Stack**

This is my personal portfolio website built with HTML, CSS, and JavaScript, hosted on **GitHub Pages**.

## Features

- **Hero Section** with typing animation showing my roles
- **About Me** with animated stats counters
- **Skills** categorized by Backend, Databases/DevOps, Angular, and React (in progress)
- **Work Experience** timeline showcasing 3 organizations
- **Achievements & Impact** cards
- **"What I Bring to the Table"** value proposition section
- **Contact form** with interactive feedback
- **Fully responsive** (mobile, tablet, desktop)
- **Smooth scroll animations** and particle effects
- **Mobile-friendly** hamburger navigation

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
